## Implementation Notes

### Pattern Consistency
- All keys follow the pattern: `Policy_Context_Detail`
- Parameterized keys use {0}, {1} for dynamic values
- Common keys (Edit, Delete, Cancel, etc.) are reused across views

### Localization Injection
All views include:
```csharp
@using Microsoft.AspNetCore.Mvc.Localization
@inject IHtmlLocalizer<SharedResource> L
```

### Usage Examples
```csharp
// Simple key
<h2>@L["Policy_List_Title"]</h2>

// With HTML
<label class="fw-bold">@L["Policy_Owner"]</label>

// Parameterized
<p>@L["Policy_Delete_Warning", Model.Title, Model.PolicyNumber]</p>

// In attributes
<button title="@L["Edit"]">...</button>
```
