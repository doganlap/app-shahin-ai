## üèóÔ∏è ARCHITECTURE PATTERNS

### EditForm Pattern (Production-Ready)
Every form now uses:
- `<EditForm Model="@model" OnValidSubmit="@HandleSubmit">`
- `<DataAnnotationsValidator />` for automatic validation
- `<InputText>`, `<InputSelect>`, etc. for typed inputs
- `<ValidationMessage>` for field-level errors
- Disabled submit button during validation/submission
- Try-catch error handling with user feedback

### LoadingSpinner Coordination
All async operations now follow:
```razor
<LoadingSpinner IsLoading="@isLoading" Message="Loading..." />
@if (!isLoading) { <!-- content --> }
```

### StatusBadge Color Mapping
Consistent status display across all pages:
- **Active/Open** ‚Üí Green
- **Pending** ‚Üí Yellow
- **Closed/Completed** ‚Üí Gray
- **Mitigated** ‚Üí Blue
- **Error/Ineffective** ‚Üí Red
