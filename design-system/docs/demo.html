Design Tokens Reference

Key Token Categories

Color Tokens

--color-primary, --color-success, --color-warning, --color-error, --color-info

Use semantic color tokens for consistent theming

Spacing Tokens

--space-padding-xs, --space-padding-sm, --space-padding-md, --space-padding-lg

8px-based spacing scale for consistent layouts

Typography Tokens

--font-size-base, --font-size-heading-sm, --font-weight-base

Scalable typography system from 12px to 40px

Component Tokens

--button-padding-vertical, --input-border-color, --card-shadow

Component-specific tokens built on semantic tokens
